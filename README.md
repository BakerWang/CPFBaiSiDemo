# CPFBaiSiDemo

> 根据百思不得姐官方OpenAPI，该项目中开发了除登录注册功能外的看帖、看评论、看图片等基本功能，还包含了获取推荐关注、推荐标签以及发段子界面添加、删除段子标签等附加功能，设置界面添加清理缓存的小功能。此外，受限于素材，未实现换肤功能。
